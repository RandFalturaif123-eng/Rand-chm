# AI for Laboratory Inventory Management

## Overview
This project is part of the Building AI course by the University of Helsinki.

The project explores how artificial intelligence can be used to improve laboratory inventory management by helping track supplies, predict needs, and support better decision-making.

## Problem
Laboratories need to manage many materials and supplies. Running out of important items or overstocking can cause delays and increase costs.

## Solution
This project proposes using AI methods to analyze inventory data and provide predictions about future supply needs.

## AI Methods
- Data analysis
- Machine learning techniques
- Prediction models

## Goals
The goal of this project is to demonstrate how AI can help laboratories manage resources more efficiently and reduce waste.
